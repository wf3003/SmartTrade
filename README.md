# SmartTrade (量策)

AI 驱动的数字货币合约自动交易系统。基于技术指标规则 + AI 决策引擎，支持多交易所、多币种、全自动运行。

## 特性

- **双引擎决策**：策略引擎（规则触发） + AI 引擎（DeepSeek 分析）
- **多币种监控**：同时监控 10+ 币种，多时间框架分析
- **全自动风控**：分批止盈 + 跟踪止盈 + 分级止损 + 账户回撤保护
- **实时仪表盘**：Web 界面展示持仓、决策、交易记录
- **多交易所**：支持 OKX、Gate.io、Binance（含模拟盘）

## 快速开始

```bash
# 安装
npm install

# 配置（编辑 .env 填入 API 密钥）
cp .env.example .env

# 启动
npm start
```

## 风控规则

| 规则 | 条件 | 动作 |
|------|------|------|
| 止损 | 亏损 ≥ -4% | 市价平仓 |
| 分批止盈 | 盈利 ≥ 10% | 平 30% 仓位 |
| 跟踪止盈 | 分批后回撤 ≥ 5% | 平剩余全部 |
| 持仓上限 | 最多 10 个 | 拒绝开新仓 |
| 每日亏损 | 累计亏损 ≥ $200 | 暂停交易 |

## 技术栈

- TypeScript + Node.js
- ccxt（交易所接口）
- DeepSeek API（AI 决策）
- SQLite（数据存储）
- Express（Web 仪表盘）

## 技术指标

每个币种综合多时间框架（1m / 5m / 15m / 1h / 1d）分析：

- **EMA 20/50** — 趋势方向
- **ADX** — 趋势强度（>25 强趋势，<18 震荡）
- **RSI 14** — 超买/超卖
- **ATR** — 波动率
- **布林带** — 支撑阻力
