# SmartTrade (量策)

AI 驱动的数字货币合约自动交易系统。基于**技术指标规则引擎** + **DeepSeek AI 方向复核**，支持多交易所、多币种、全自动运行。

## 特性

- **六类行情分级**：强趋势多/空 → 弱趋势多/空 → 震荡偏多/空 → 纯震荡，每级不同杠杆和入场策略
- **动态入场**：靠近均线做反弹/回踩（高置信度），远离均线追入（低置信度）
- **AI 方向复核**：DeepSeek 每周期校验策略信号方向，不一致时跳过
- **方向分散**：同方向最多 5 个持仓，防单边集中风险
- **超涨超跌保护**：价格偏离 3ATR + RSI 极端时，禁止同方向开仓并预警
- **全自动风控**：ATR 动态止损（5-10%）+ 实际价格跟踪止盈 + 账户回撤保护
- **实时仪表盘**：Web 界面展示持仓、决策、交易记录
- **多交易所**：支持 OKX、Gate.io、Binance（含模拟盘）

## 快速开始

```bash

# 配置（编辑 .env 填入 API 密钥）

```

## 行情分类与策略

| 行情类型 | ADX 范围 | 杠杆 | 入场方式 | 置信度阈值 |
|---------|---------|------|---------|-----------|
| 强趋势多/空 | ≥40 | 6x (基数) | 回踩/反弹 + 追入 | 35% |
| 弱趋势多/空 | 25-40 | 4x | 回踩/反弹 + 追入 | 40% |
| 震荡偏多/空 | 18-25 | 2x | 仅回踩/反弹（禁止追入） | 55% |
| 纯震荡 | <18 | 不开仓 | — | — |

## 风控规则

- **止损**：ATR × 1.2 × 杠杆，范围 [5%, 10%]（低波动 5%，高波动 10%）
- **跟踪止盈**：基于实际价格（pnlPct ÷ 杠杆），涨 ≥1.5% 激活，保底距离 0.6%，涨 ≥3% 缩窄到 0.5%
- **同方向限制**：最多 5 个同方向持仓
- **超跌/超涨**：偏离 3ATR + RSI<30/>70，禁止同方向开仓
- **持仓上限**：最多 20 个
- **每日亏损**：累计亏损 ≥ $200 暂停交易
- **账户止损**：权益 ≤ $500 平所有仓
- **止损冷却**：止损后该币种暂停交易 30 分钟

## 技术栈

- TypeScript + Node.js + tsx watch
- ccxt（交易所接口）
- DeepSeek API（AI 方向复核）
- SQLite（数据存储）
- Express（Web 仪表盘）

## 技术指标（1h 框架）

每个币种综合多时间框架（1m / 5m / 15m / 1h / 1d）分析：

- **EMA 20/50** — 趋势方向
- **ADX** — 趋势强度（≥40 强趋势，<18 纯震荡）
- **RSI 14** — 超买/超卖（>70 超涨，<30 超跌）
- **ATR** — 波动率（用于动态止损和入场带）
